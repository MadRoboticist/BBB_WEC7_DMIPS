Detailed Description

A program to perform G.711 encoding/decoding modified from: https://github.com/escrichov/G711 to support Windows CE 7 on a Beaglebone Black.

Author
David Pasley

The program was modified to handle PCM and G.711 headers. If doing G.711->PCM conversion, a-law or u-law is determined by information in the header.
